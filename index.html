pokertable — Support

Fast, fair dealing for Texas Hold’em home games.

How to use

  1. Open the app on two or more devices.
  2. Tap Host on one device to create a 4‑character code.
  3. On the others, enter your name and tap Join with that code.
  4. The host taps Deal to progress: hole cards → flop → turn → river.
  5. Optionally set an iPad as the board display. Phones only show their own two cards.

FAQ

Does this include chips or betting?
No. It only deals cards to keep games moving.

What data do you collect?
Just a player name and a random device ID to sync a session. No ads or tracking.

It won’t join a table.
Check your internet connection and that the game code hasn’t expired. Ask the host to start a new hand.
